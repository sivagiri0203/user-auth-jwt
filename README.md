User Authentication using JWT
Project Description

This is a simple Node.js authentication API using JWT (Bearer Token).
Users can register, login, and access a protected route using a JWT token.

Tech Stacks used

Node.js
Express.js
MongoDB (Mongoose)
JWT

Postman

API Endpoints
Register
POST /api/users/register
Login
POST /api/users/login
Get Profile (Protected)
GET /api/users/profile
